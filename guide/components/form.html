FORM

Overview 본 프로젝트에서 사용되는 폼의 종류와 형태를 정리합니다.

입력폼

LAYOUT
  • 폼 타입 테이블(table.tbl-form)을 사용하여 폼 입력항목을 배치합니다.
  • 테이블을 4개의 컬럼(th, td)로 나눠서 첫번째, 세번째 컬럼(th)에는 입력항목의 제목을 두번째, 네번째 컬럼(td)에는 입력창을 배치합니다.
  • table.tbl-form에 설정된 min-width는 1,000px입니다.
  • 기본 th width는 200px 입니다.
  • 기본 row height : 49px (border-bottom 1px 포함)

(입력폼 소제목 강조컬러)
div.input-wrap > input

div.input-wrap > input
div.input-wrap > input
예시
  • 필수입력항목에는 label에 .requried 클래스를, input에 required="required" 속성을 입력합니다.
  • placehoder(select의 경우는 data-placeholder)값은 validation에서 message값으로도 사용됩니다.

검색폼

LAYOUT
  • Bootstrap의 .col-* 칸을 나누는 개념을 사용하여 class를 생성하였습니다. 해당 클래스들을 이용하여 폼 입력항목을 배치합니다.
  • .search_form_wrap의 min-width는 1,000px, .search-form의 min-width는 900px이고, 각 .row의 높이는 45px입니다.
  • .search_form_inner을 좌우로 나눠서, 왼쪽(.search-form)에 검색조건 input들을, 오른쪽에 검색 버튼을 배치합니다.
  • .search_form_inner 태그는 .form_wrap 안쪽에 있어야 합니다.
