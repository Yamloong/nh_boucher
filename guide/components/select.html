SELECT

기본 형태 1

LABEL + SELECT

  • * custom select는 select2를 style 변경하여 사용함. 기본 사용법은 https://select2.org/을 참조하시기 바랍니다.
기본 형태 1
  • select tag의 class에 "select2" class가 들어있으면 모두 select2 형태로 변환됩니다. (pubUtil.js에 관련 기능이 구현되어 있습니다.)
  • 필수입력인 경우, data-placeholder를 반드시 입력해야 합니다.
  • 첫번째 옵션에 반드시 빈 옵션(<option></option>)을 두어야 합니다. (단, 검색조건으로 사용할 경우는 예외)
  • 필수입력항목(required="required")인 select가 아닌 경우, 선택된 옵션 뒤에 [X] 버튼이 노출됩니다.
  • select를 readonly하고자 할때는 같은 이름의 input[type=hidden]을 추가하여 값을 전달하고, select는 disabled="disabled" 합니다.
  • 검색조건으로 select2를 사용하고자 할 때는 아래 '검색' 예시처럼 1)required="required" 2)첫번째 옵션에 '전체' 형태로 사용하시기 바랍니다.
    required로 안하면 선택창 뒤에 [X]가 노출됩니다.
