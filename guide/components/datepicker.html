DATE PICKER

Overview jQuery UI 등 3rd party 제품 기반의 component 예시와 사용방법입니다.

DatePicker

기본형태 1
  • DatePicker를 설치할 input에 .datepicker class를 추가 합니다.
기본 형태 2 왼쪽 레이블 + 레이블 width에 따라 input width 가변
기간입력 + 옵션버튼
  • 기간입력이면 입력창을 아래 예시처럼 .input_period_wrap으로 감싸야 합니다.
  • 기간선택 단축버튼은 '금일'처럼 setInputPeriod()에 넘겨받을 시작일, 종료일 입력창을 파라미터로 넘겨야 합니다.
  • 그외 단축버튼처럼 파라미터를 안넘길 경우, 부모(.input_period_wrap)에서 .from을 시작일로, .to를 종료일로 처리합니다.
  • div.button-wrap (기간선택 단축버튼 그룹)을 제거하여 기간입력창만 사용할 수도 있습니다.
