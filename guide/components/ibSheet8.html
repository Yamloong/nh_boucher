IB Sheet를 사용한 데이터 그리드의 예시와 사용방법입니다.
Row Height (border 제외) : Header-45px / Data-45px / Footer-45px
홈페이지 : https://www.ibsheet.com/

예시

  • ID값은 자유롭게 부여하시고 사용하시되, .grid-area 안에 그리드를 생성해주세요.
  • 해당 소스는 IBSheet에 적용한 디자인을 확인하기 위한 소스입니다. 개발용 소스는 IBSheet API문서를 참고해주세요.
  • 버튼은 컴포넌트/button.html 가이드 파일에 명시한 그대로 넣어주시면 됩니다. 또는 아래의 소스 197번줄을 참고해주세요.
  • pagination 관련하여서는 html에서 구현 불가능한 환경이오니, class명은 ibsheet7.html파일을 참고해주시고 기능 구현은 API문서를 참고해주세요.
