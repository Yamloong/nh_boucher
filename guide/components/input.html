응용 형태 숫자 입력 > .unit(단위)
  • 숫자만 입력받을 때 input type=number로 함. 단, 1000단위 comma를 찍어주고자 할때는 type=text(모바일 일때는 tel)로 할 것.
  • 아래 예시 '금액' 입력처럼 입력과 동시에 ','를 보여주고자 할때는 onkeyup 이벤트에 numberLocaleString()을 호출할 것.
  • 아래 예시 '퍼센트' 입력처럼 소숫점 있는 숫자를 입력받을 때는 반드시 onfocusout 이벤트에 numberLocalString()을 호출할 것.
  • numberLocaleString()함수는 pubUtil.js에 있습니다.
