rMateGrid

rMateGrid를 사용한 데이터 그리드의 예시와 사용방법입니다.
Row Height (border 제외) : Header-32px / Data-28px / Footer-28px
홈페이지 : http://demo.riamore.net

예시

  • 그리드가 삽입될 ID값은 고정으로 #gridHolder를 사용하시되 #gridHolder 안에 그리드를 생성하실 ID(ex. grid1, grid2)는 자유롭게 사용하시면 됩니다.
  • 해당 소스는 rMateGrid에 디자인을 적용하기 위한 퍼블용 소스입니다. 개발용 소스는 rMateGrid에   API문서 를 참고해주세요.
  • 그리드 cell 안에서 사용할 버튼은   컴포넌트/button.html   가이드 파일에 명시한 그대로 넣어주시면 됩니다.
